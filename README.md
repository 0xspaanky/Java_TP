# TP Java – Mini Calculatrice (Menu Console)

## Description

Ce TP est une petite application Java en mode console qui joue le rôle de **mini calculatrice**.  
L’utilisateur choisit une option dans un menu, saisit les nombres demandés, et le programme affiche le résultat.

Fonctionnalités :

1. **Addition** de deux entiers  
2. **Multiplication** de deux entiers  
3. **Moyenne** d’une liste de valeurs entières  
4. **Recherche du maximum** dans une liste d’entiers  
5. **Quitter** le programme

Tout se fait via le clavier et l’affichage console.

---

## Structure du projet

Dans le dossier du TP, on trouve :

- `Test.java` : fichier **source Java** contenant la classe principale (avec la méthode `main`).
- `Test.class` : fichier **compilé** généré automatiquement par le compilateur Java.

> Remarque : selon l’éditeur utilisé, le nom de la classe peut être `Main` ou `Test`, mais l’idée est la même : c’est la classe contenant la méthode `main`.

---

## Prérequis

- Java installé (JDK 8 ou plus récent)
- Un terminal / invite de commande
- (Optionnel) Un IDE comme IntelliJ IDEA, Eclipse ou VS Code

---
